# PYINSTAMATION

> Easy to use instagram bot written in python 3


## Features

* Upload pictures
* Follow/unfollow
* Like, comment by tags


## Requirements

```
python 3
```


## Instalation

`pip install -r requirements.txt`


## Configuration

Provide credentials in the `config.yaml`.

Edit variables in `config.py`


## Usage

`./insta_cli.py`


We told you it was easy.
